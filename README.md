# PDR Website

Live site: [https://pdr.phfk.college/](https://pdr.phfk.college/)

Repository: [https://github.com/sashik117/pdr-test](https://github.com/sashik117/pdr-test)

## Project Overview

This project is a PDR learning platform with:
- Frontend: Next.js + React + TypeScript
- Backend: Node.js + Nitro + TypeScript
- Database: MongoDB

## Local Development

- Frontend: `frontend/`
- Backend: `backend/`
- Admin panel: `admin/`
- Parser scripts: `scripts/`

See backend deployment notes in [`backend/docs/DEPLOY.md`](backend/docs/DEPLOY.md).
